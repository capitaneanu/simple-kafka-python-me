# A Simple Producer

There's a complete [walkthrough video here][youtube-producer]. To run the code in this repo:

```sh
# Set up an environment.
cd send_to_kafka
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt

# Run the producer.
python3 main.py
```
